# Amazon Web Scraping Project

## Overview
This project demonstrates how to collect product data from an Amazon product page using Python web scraping techniques. The script extracts key information such as product title and price and stores the data in a structured format for further analysis.

Web scraping allows analysts to gather publicly available data from websites automatically for analysis, monitoring, and automation purposes. This project showcases practical data collection skills relevant to data analytics and automation workflows.

## Project Objective
The objective of this project is to:
- Extract product data from an Amazon webpage
- Clean and structure the extracted information
- Store the data for analysis and tracking
- Demonstrate automation and data collection using Python

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
- CSV


## Features
- Connects to an Amazon product webpage
- Extracts product title and price
- Cleans and structures scraped data
- Stores data in a CSV file
- Demonstrates automated data collection

## Project Workflow
1. Send a request to the Amazon product webpage.
2. Retrieve the HTML content of the page.
3. Parse the HTML using BeautifulSoup.
4. Extract relevant data such as:
   - Product title
   - Product price
5. Clean and format the extracted data.
6. Save the data to a CSV file for analysis.
